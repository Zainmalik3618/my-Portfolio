Add CalmConnect AI screenshots in this folder.

Recommended image names:

- dashboard.png
- chat.png
- mood-tracker.png
- journal.png

Then add them to `src/data/projects.js` like this:

```js
screenshots: [
  { src: '/projects/calmconnect/dashboard.png', alt: 'CalmConnect AI dashboard' },
  { src: '/projects/calmconnect/chat.png', alt: 'CalmConnect AI chat support' },
]
```
