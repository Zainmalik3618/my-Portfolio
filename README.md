# Zain Malik Portfolio

A frontend-only portfolio website built with React, Vite, Tailwind CSS, Framer Motion, and Lucide React.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Installation

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Editing Projects

Project cards are powered by `src/data/projects.js`. Add or edit project entries there without changing the Projects components.

## Editing Skills

Skills are powered by `src/data/skills.js`. Add or remove skills there and the Skills section updates automatically.

## Contact Form

The contact form is frontend-only. Add your Formspree endpoint or EmailJS logic in `src/components/Contact.jsx` where the inline comment is placed.
